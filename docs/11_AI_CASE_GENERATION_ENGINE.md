## الهدف

تحويل CASEFORGE Studio من محرر يدوي إلى مساعد ذكي يستطيع إنشاء قضية كاملة
خلال دقائق. الأدمن لا يبدأ من صفحة فارغة، بل يصف فكرة القضية، والنظام
يولد النسخة الأولى كاملة.

## الواجهة

داخل CASEFORGE Studio يوجد زر كبير في الصفحة الرئيسية:
```
+-------------------------------------------+
|           ✨ Generate AI Case              |
|                                           |
| Create a complete investigation           |
| using artificial intelligence             |
+-------------------------------------------+
```
عند الضغط عليه يبدأ معالج (Wizard).

## الخطوة الأولى: اختيار نوع الجريمة
Murder / Missing Person / Kidnapping / Theft / Fraud / Cyber Crime /
Poisoning / Corporate Crime / Custom

## الخطوة الثانية: مكان الجريمة
فندق، مستشفى، شركة، جامعة، فيلا، مطعم، سفينة، متحف، بنك، مركز تجاري —
أو مكان مخصص.

## الخطوة الثالثة: مستوى الصعوبة
Beginner / Intermediate / Advanced / Expert
يؤثر تلقائيًا على: عدد الأدلة، عدد المشتبه بهم، عدد الوثائق، عدد الأسئلة،
مدة التحقيق.

## الخطوة الرابعة: تفاصيل القضية
الأدمن يكتب وصفًا بسيطًا، مثال:
"أريد قضية قتل داخل فندق فاخر في دبي. يوجد أربعة مشتبه بهم. القاتل هو
مدير الفندق، لكن الأدلة يجب أن تجعل اللاعب يشك في الطاهي حتى النهاية."

## الخطوة الخامسة: زر التوليد
✨ Generate Case

## ماذا يحدث في الخلفية؟

محرك الذكاء الاصطناعي يبدأ بتوليد العناصر بالترتيب:
1. القصة (خلفية القضية، الضحية، سبب الجريمة، تسلسل الأحداث)
2. الضحية (الاسم، العمر، الوظيفة، الصورة، تاريخ ووقت وسبب الوفاة)
3. المشتبه بهم (الاسم، الصورة، العمر، الوظيفة، العلاقة، الدافع، السلوك،
   مدى الاشتباه)
4. الشهود (أسماء، إفادات، تناقضات، مستوى الموثوقية)
5. الأدلة (بصمات، سلاح، هاتف، كاميرات مراقبة، رسائل، DNA، آثار أقدام،
   ألياف، تقارير مخبرية)
6. الوثائق (تقرير الشرطة، الطب الشرعي، مسرح الجريمة، البصمات، DNA،
   السموم، الأدلة، محاضر الاستجواب)
7. الخط الزمني (مثال: 18:20 وصول الضحية → 19:05 بدء الجدال → 19:40
   انقطاع الكهرباء → 19:47 طلقة نارية → 20:03 وصول الشرطة)
8. الأسئلة (من هو القاتل؟ ما الدافع؟ أي دليل الأهم؟ رتب الأحداث؟ اختر
   السلاح؟)
9. الحل — لا يظهر للاعب، يُحفظ في solution.json (القاتل، الدافع، السلاح،
   ترتيب الأحداث، الإجابات الصحيحة)

## مراجعة الأدمن

بعد التوليد لا يتم التصدير مباشرة. تظهر صفحة مراجعة، يمكنه: تعديل النصوص،
حذف الأدلة، إضافة مشتبه، تغيير الصور، إعادة توليد عنصر واحد فقط (مثال:
Regenerate Witnesses أو Regenerate Evidence) بدون إعادة إنشاء القضية
كاملة.

## توليد الصور

لكل عنصر يوجد زر: Generate Image أو Replace Image — يستخدم وصفًا (Prompt)
مناسبًا لكل نوع (مشتبه، دليل، مسرح جريمة، ضحية، شاهد).

## التحقق من الجودة

قبل السماح بالتصدير، يجري النظام فحصًا: هل يوجد حل؟ هل عدد الأدلة مناسب؟
هل يوجد تناقض منطقي؟ هل كل سؤال له إجابة؟ هل جميع الصور موجودة؟ هل كل
الوثائق مكتملة؟ إذا وجد نقصًا، يعرض قائمة بالمشكلات.

## التصدير

بعد النجاح: Generate Case Package → ينتج DX021.casepack جاهز للبيع.
