# Overview

CASEFORGE Studio supports three AI generation modes. Each mode targets a
different type of creator.

---

# MODE 1 — QUICK GENERATE

Purpose: Generate a complete investigation in less than 2 minutes.
Designed for users who only have a rough idea.

User Interface
```
+------------------------------------------------------+
|              ✨ QUICK GENERATE                       |
|------------------------------------------------------|
| Crime Type        [ Murder ▼ ]                        |
| Difficulty        [ Intermediate ▼ ]                  |
| Location          [ Luxury Hotel ▼ ]                  |
| Short Idea        "A businessman is found dead        |
|                     inside room 308."                  |
| Estimated Play Time [ 90 Minutes ▼ ]                   |
|              [ GENERATE CASE ]                        |
+------------------------------------------------------+
```

What happens internally: the AI automatically generates Case title, Story,
Victim, Crime Scene, Suspects, Witnesses, Timeline, Evidence, Police
Reports, Autopsy, Questions, Solution, Cover Image Prompt. No additional
user input required.

Expected Time: 30–90 seconds
Best For: Beginners, fast content creation, prototype investigations

---

# MODE 2 — GUIDED GENERATE

Purpose: Allow the creator to control every important aspect while AI
fills the details.

Wizard Steps
1. Crime Type — Murder / Kidnapping / Missing Person / Theft / Fraud /
   Cyber Crime
2. Location — Luxury Hotel / Hospital / Museum / University / Office /
   Villa / Restaurant / Bank / Airport / Custom
3. Difficulty — Beginner / Intermediate / Advanced / Expert
4. Number of Suspects — 2 / 3 / 4 / 5 / 6
5. Victim Information — Generate Automatically or Enter manually
6. Story Theme — Family Conflict / Business Rivalry / Jealousy /
   Inheritance / Political / Organized Crime / Serial Killer / Random
7. Twist — Wrong Suspect / Hidden Twin / Fake Death / Inside Job /
   Corrupt Officer / Double Murder / None
8. Desired Investigation Length — 30 / 60 / 90 / 120 / 180 Minutes
9. Generate — the AI creates the investigation respecting every choice

Generated Content: Victim, Suspects, Witnesses, Evidence, Timeline, Crime
Scene, Police Documents, Questions, Solution, Image Prompts, Cover Prompt,
Certificate Title, Difficulty Score

Expected Time: 2–5 minutes
Best For: Professional creators, premium investigations, commercial
releases

---

# MODE 3 — MANUAL BUILD

Purpose: Create every part manually without AI.

Dashboard: General Information, Victim, Crime Scene, Evidence, Witnesses,
Suspects, Timeline, Documents, Questions, Solution, Preview, Export

Each section is empty. The creator builds everything manually.

Example flow: Click "Add Suspect" → Upload Photo → Enter Name →
Occupation → Relationship → Motive → Background → Save. Repeat until
finished.

Advantages: Complete creative control. Perfect for real investigations.
Ideal for educational cases.

## AI Assistance Inside Manual Mode

Even in Manual Mode every field has an AI helper.

- Story Field: Generate / Improve / Rewrite / Expand / Translate
- Evidence: Generate Fingerprint Report / Generate DNA Report / Generate
  Police Report / Generate Witness Statement / Generate Crime Scene Notes
- Images: Generate Portrait / Generate Evidence / Generate Crime Scene /
  Generate Building / Generate Map
- Timeline: Generate Timeline / Improve Timeline / Fix Logical Errors
- Questions: Generate Questions / Balance Difficulty / Create Trick
  Question
- Solution: Check Consistency / Detect Plot Holes / Validate Investigation

# Project owner's closing recommendation

If CASEFORGE Studio is to become a standout product, AI should not be a
single button — it should be a helper present throughout the app. Any page
the user opens should offer Generate / Improve / Rewrite / Expand /
Validate actions, making AI a partner in writing, improving, and reviewing
every part of a case rather than just a generator.
